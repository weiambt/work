- 算法：[260. 只出现一次的数字 III](https://leetcode.cn/problems/single-number-iii/)

  没做出来，两个元素只出现1次，其余出现2次

  异或和、根据异或和的位划分（lowbit）

  ```
  func singleNumber(nums []int) []int {
      two := 0
      for _,x := range nums{
          two ^= x
      }
      lowbit := two & -two
      a := 0
      b := 0
      for _,x := range nums{
          if x & lowbit!=0{
              a ^= x
          }else{
              b ^= x
          }
      }
      return []int{a,b};
  }
  ```
